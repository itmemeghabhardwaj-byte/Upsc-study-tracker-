# 📚 UPSC Study Tracker

A comprehensive, feature-rich web-based study tracker specifically designed for UPSC aspirants. Track your daily study progress, monitor subject-wise completion, analyze test scores, and manage your revision schedule effectively with a beautiful UI and dark mode support.

## ✨ Key Features

### 📊 Dashboard
- **Real-time Statistics**: Today's study hours, total study days, average daily hours, and GS completion percentage
- **Weekly Study Chart**: Visual bar chart showing study hours across the week
- **Subject Distribution**: Pie chart showing time distribution across subjects

### 🎯 Daily Targets
- Set daily study hour targets (1-24 hours)
- Create and manage today's study targets
- Visual list of pending targets
- Quick task addition with descriptions

### ⏱️ Study Hours Logging
- Log study sessions with date, subject, hours, and detailed notes
- Track multiple subjects: GS-I, GS-II, GS-III, GS-IV, Optional, Revision, Current Affairs
- Complete study history with sorting and deletion
- Cumulative statistics updated in real-time

### 📖 GS Progress Tracking
- Individual progress tracking for all four General Studies papers
- Interactive progress sliders (0-100%)
- Visual progress bars with smooth animations

### 🔬 Optional Subject Progress
- Select from 6 optional subjects
- Track progress for your chosen optional subject
- Dynamic progress visualization

### 🔄 Revision Tracker
- Schedule revision topics with customizable intervals
- Automatic next-revision date calculation
- Track revision status (pending/completed)

### 📈 Test Score Analytics
- Log test scores and percentiles
- Visualize score trends with line charts
- Comprehensive test records table

### 🌙 Dark Mode
- Toggle between light and dark themes
- Eye-friendly dark mode for comfortable studying
- Theme preference saved automatically

### 💾 Data Management
- All data saved locally in browser's localStorage
- No backend required - completely offline capable

### 📱 Responsive Design
- Works on desktop, tablet, and mobile devices
- Touch-friendly interface

## 🚀 Getting Started

### Option 1: GitHub Pages Deployment
1. Go to **Settings** → **Pages**
2. Select **Source**: `main` branch
3. Your tracker will be available at: `https://yourusername.github.io/Upsc-study-tracker-`

### Option 2: Local Usage
1. Clone the repository
2. Open `index.html` in your browser
3. Start tracking!

## 🎨 Customization

### Change Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
}
```

### Modify Subjects
Edit subject options in `index.html` select elements
